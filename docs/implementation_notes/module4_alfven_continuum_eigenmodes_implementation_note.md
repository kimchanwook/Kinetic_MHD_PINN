## Current status

A conventional self-adjoint two-harmonic reduced shear-Alfvén benchmark is implemented independently in Python and MATLAB. Python tests and reference generation pass. MATLAB source and tests are included but were not run in the build environment because MATLAB is unavailable. The companion physics note is a 77-page modular textbook chapter, version 2.0, and explicitly distinguishes the verified avoided-crossing benchmark from a predictive or strongly localized stellarator TAE. No PINN is included yet.
